Standard Normal Distribution

Why This Concept is Added:

  • Standardization: Any normal distribution can be transformed into the standard normal distribution by converting values to z-scores. The formula for a z-score is:
    Z = (X - μ) / σ
    Here, Z represents the number of standard deviations a value X is from the mean μ.
  • Universal Reference: The standard normal distribution always has a mean (μ) of 0 and a standard deviation (σ) of 1. This makes it easier to use z-tables or software tools to calculate probabilities.
  • Probability Calculations: The standard normal distribution has a predefined probability density function (PDF):
    f(z; 0, 1) = (1 / √2π) e^(-z² / 2)
    Probabilities and cumulative areas under the curve can be derived using this single formula for any z-score.
  • Applicability: Standardizing values allows you to:
    • Compare datasets with different scales.
    • Compute probabilities and critical values in hypothesis testing.
    • Evaluate percentiles in a consistent manner.

Including this concept ensures learners understand the transition from a general normal distribution N(μ, σ²) to the standard normal distribution N(0, 1), which is a foundation for many statistical applications.

Random Variable

A random variable is a numerical value assigned to the outcomes of a random experiment. It maps the outcomes of an experiment to numerical values.

1. Discrete Random Variable

A discrete random variable takes specific, countable values. It is used when the sample space consists of distinct, separate outcomes.

Example: Tossing a fair coin twice.

  • Sample Space: {HH, HT, TH, TT}
  • Define a random variable X as the number of heads observed:
    • X(HH) = 2
    • X(HT) = 1
    • X(TH) = 1
    • X(TT) = 0
  • Possible values of X: {0, 1, 2}

Explanation: The outcomes (HH, HT, TH, TT) are distinct, and the random variable X assigns numerical values (0, 1, or 2) to them. These values are countable.

2. Continuous Random Variable

A continuous random variable takes any value within a given range. It is used when the sample space is uncountable, such as measurements of height, weight, or time.

Key Difference:

  • Discrete Random Variable: Deals with countable outcomes (e.g., the number of heads in coin tosses).
  • Continuous Random Variable: Deals with measurable, continuous outcomes (e.g., age, weight, or temperature).
